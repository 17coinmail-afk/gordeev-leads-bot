# Агрегатор фриланс-заказов (Telegram-бот)

Полноценный SaaS для фрилансеров. Собирает заказы с 3 бирж, анализирует AI, фильтрует по навыкам и присылает в Telegram. Монетизация через СБП с автоактивацией.

## Фишки

- **3 биржи**: FL.ru, Freelance.ru, Kwork (через Playwright)
- **AI-анализ заказов**: GPT-4o-mini оценивает ТЗ и бюджет
- **Умные рассылки**: дайджесты, антиспам, inline-кнопки
- **Оплата СБП**: QR-код + автоактивация через email
- **Pro-подписка**: фильтрация по тарифам
- **Антифлуд**: защита от спама команд
- **Админ-панель**: рассылки, бан/разбан, статистика
- **Веб-дашборд**: метрики в реальном времени
- **Автоочистка**: удаление старых данных
- **Бэкапы**: скрипт резервного копирования БД

## Команды бота

### Пользовательские
| Команда | Описание |
|---------|----------|
| /start | Регистрация |
| /help | Полная справка |
| /setkeywords | Фильтры через запятую |
| /setbudget | Минимальный бюджет |
| /template | Шаблон отклика |
| /subscribe /unsubscribe | Уведомления |
| /stats | Личная статистика |
| /ref | Реферальная ссылка |
| /pay | Купить Pro (QR-код) |
| /pro | Статус подписки |
| /status | Мои настройки |
| /check | Проверить заказы |

### Админские
| Команда | Описание |
|---------|----------|
| /admin | Панель управления |
| /activate user_id days | Активировать Pro |
| /broadcast текст | Рассылка всем |
| /ban user_id | Заблокировать |
| /unban user_id | Разблокировать |

## Быстрый старт (локально)

```bash
pip install -r requirements.txt
playwright install chromium
# Создай .env по шаблону .env.example
python main.py
```

## Настройка (.env)


## Веб-дашборд

После запуска открывается на `http://IP:8080`

Показывает:
- Количество пользователей
- Активных подписчиков
- Pro-пользователей
- Заблокированных
- Заказов в базе
- Уведомлений за день

## Бэкап

```bash
# Ручной бэкап
bash deploy/backup.sh

# Автоматический через cron (каждый день в 3:00)
0 3 * * * /opt/freelance-bot/deploy/backup.sh
```
